# Power BI Sales Analytics Dashboard

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze retail sales performance and customer insights.

## Key Features
- KPI analysis: Total Sales, Profit, Profit Margin, Average Order Value
- Sales trends over time
- Product, customer, and regional performance analysis
- User-friendly navigation and synced slicers

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
